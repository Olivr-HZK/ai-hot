- 在对话中按平台列出所有通过内容：

  - 平台
  - 推送对象：`产品` / `UA` / `ALL`
  - 主产品：Evoke / Toki / Kavi / Avatar / none
  - 链接

  - 各平台输入数量
  - 进入模型审核数量
